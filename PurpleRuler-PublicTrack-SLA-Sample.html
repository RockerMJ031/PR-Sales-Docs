SLA · Standardized Learning Assessment

同年龄段分位水平评估 + 5 分钟家长沟通诊断脚本

SLA = Standardized Learning Assessment（标准化学习评估）—— 教育学评估概念，用于判断孩子在同年龄段同学中的分位水平（前 5% / 前 10% / 中位 / 后 30% / 后 50%）。英美公立学校系统广泛使用。

Purple Ruler 公立路径用 SLA 帮销售/教学顾问做两件事：① 让老师跟家长沟通时显得专业——给出可量化的分位判断，而不是模糊的"还不错";② 把分位映射到正确的课时包——基础/标准/冲刺三档不再凭感觉选。这一页就是 5 分钟从家长答案推导分位、再推导推荐课包的诊断工具。

三条评估线索 · 销售/顾问必懂

家长说"我家孩子英语挺好的"是没有信息量的。SLA 评估靠下面三条客观线索拼出分位。

线索 1 · 标准化考试
KET / PET / FCE 等级
剑桥少儿英语标化体系是国内公认的硬指标。

关键不是"过没过"，而是"什么等级过的"：
  • Pass with Distinction（卓越通过）= 分位前 10%
  • Pass with Merit（良好通过）= 中上分位
  • Pass（通过）= 中位
没考过 KET 之前 → 用 YLE Flyers / 雅思少儿 / 学而思测评等级当替代锚点。
线索 2 · 班级排名 + 城市修正
绝对排名 ÷ 城市层级
班级排名是高频信号但必须按城市层级修正：
  • 北京海淀 / 上海徐汇 班级前 3 ≈ 全国前 5%
  • 新一线（成都/杭州/武汉）班级前 3 ≈ 全国前 15%
  • 二三线市区班级前 3 ≈ 全国前 30%
  • 县城/乡镇班级前 3 ≈ 全国中位
家长最容易高估的就是这一条——"我们班第一"在郫县下属镇 ≠ 在海淀。
线索 3 · 区域参考
EF 中国英文熟练度地图
EF Education First 每年发布的中国英文熟练度地图，按城市分级。

用法：拿家长所在城市的 EF 等级（高 / 中 / 低）做背景修正——同样班级排名第 3，在 EF 高熟练度城市（如上海）和低熟练度城市（如某地县城）的全国分位差很大。

销售话术：引用 EF 数据让家长信服"我不是在贬低你孩子，我是在给你一个客观的全国坐标"。
DIAGNOSTIC TOOL · 5 分钟评估 SLA 分位
家长沟通诊断脚本
销售/教学顾问在咨询前 5 分钟，不问"孩子英语怎么样"（家长说不准），而是问下面 5 个具体问题，根据答案拼出 SLA 分位，对应到正确课时包。

五题诊断脚本 · 销售/顾问必背

Q1 "孩子考过 KET / PET / FCE 吗？是 Pass、Merit 还是 Distinction？"
为什么问：剑桥标化等级是 SLA 评估的最硬锚点。Distinction = 前 10% 分位的客观证据。
分流：没考过 → 先用 YLE Flyers / 学而思测评替代 · KET Pass → 中位 · KET Distinction / PET Pass → 中上 · PET Distinction / FCE Pass → 前 15% · FCE Distinction → 前 5%
Q2 "孩子在班里英语排第几？学校在哪个城市的什么区？"
为什么问：班级排名是高频信号但必须做城市层级修正——海淀班级第 3 和县城班级第 3 不是一回事。
分流：一线核心区（海淀/徐汇/天河）前 3 → 前 5% · 新一线市区前 3 → 前 15% · 二三线前 3 → 前 30% · 县城/乡镇前 3 → 中位
Q3 "孩子参加过外研社 / 希望之星 / 21 世纪杯这类英语竞赛吗？最高拿过什么奖？"
为什么问：竞赛获奖能直接证明分位。家长不一定主动说，要追问。
分流：没参加过 → 中性信号 · 区级奖 → 前 30% · 市级奖 → 前 15% · 省级 / 国家级奖 → 前 5%
Q4 "孩子学校有外教课吗？是英美母语外教还是其它国家的？每周几节？"
为什么问：评估孩子累计真实英语输入小时数——SLA 分位高的孩子背后通常有持续的高质量输入。
分流：无外教课 → 输入弱 · 非英美外教（菲/印）→ 中等 · 英美外教 1-2 节/周 → 中上 · 英美外教 3+ 节/周 → 前 15%
Q5 "孩子参加过国际课程 / 英美夏校 / 海外研学吗？或者家庭有海外背景吗？"
为什么问：拿到分位的额外加成因子——有这些经历的孩子通常分位被低估。也判断家庭对国际化路径的接受度（影响课包升级空间）。
分流：都没有 → 标准分位 · 有夏校/研学 → 分位 +1 档（如中位 → 中上）· 国际课程在读 → 直接进入冲刺包 · 父母英美留学背景 → 家庭决策窗口大，可推荐高客单

SLA 沟通的三个常见误区 · 销售必懂

误区 1
家长说"我家孩子英语挺好的"
这是零信息量的回答，不要顺着接。
销售回应：用 SLA 框架反问——"挺好"在同年龄段里能到前 30% 还是前 5%？我们用三条线索（标化等级、班级排名、外教时长）来给出客观分位。
误区 2
"我们班第一名"= 全国前列
班级排名必须乘以城市层级修正系数。
销售回应：引用 EF 中国英文熟练度地图——"在 X 城市 X 区，班级第 1 大约对应全国前 X%。这是一个不带情绪的客观坐标。"
误区 3
"考过 KET 就行了"
关键不是过没过，而是什么等级过的。
销售回应：Pass / Merit / Distinction 对应的全国分位差距非常大——同样"考过 PET"，Distinction 是前 10%，Pass 只是中位偏上。
销售必懂的三句话：
① "'孩子英语好'是没有信息量的——SLA 给我们一个客观分位"——把家长拉回到框架。
② "5 个问题告诉我您孩子在同龄人中的分位"——立专业可信度。
③ "分位决定课包，不是凭感觉"——把"贵不贵"转移到"对不对症"。
